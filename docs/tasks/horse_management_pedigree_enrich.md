В нём есть foals - там отдаются все потомки
Проблема в том, что у всех потмоков нам необходимо знать ТОЛЬКО первое поколение производителей
# Что нужно сделать
Давай приведём DTO при переданном pedigree параметре к следующему виду:
```json
{
	"horse_field_1": ...,
	"horse_field_2": ...,
	"horse_field_3": ...,
	"pedigree": {
		"sire": {...},
		"dam": {...},
		"foals": [
			{
				"foal_field_1": ...,
				"foal_field_2": ...,
				"foal_field_3": ...,
				"parents": {
					"sire": {
						"id": ...,
						"name": ...
					},
					"dam": {
						"id": ...,
						"name": ...
					}
				}
			},
			{...}
		]
	}
}
```
Мы тут только у foals добавляем parents. Нужны только UUID и кличка
