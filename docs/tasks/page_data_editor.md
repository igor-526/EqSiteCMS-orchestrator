# Контекст
Следующие модели:
- breeds
- coat_color
- horse_service
- prices
Имеют колонку `page_data`
В этой колонке хранится HTML дополнительного содержания страницы
На фронте на данный момент нет никаких возможностей редактировать это содержимое
Нам необходимо внедрить данный функционал на фронт CMS в следующие места

| Страница | Раздел        | Место                          |
| -------- | ------------- | ------------------------------ |
| /horses  | Породы        | Колонка "действия", 2-я кнопка |
| /horses  | Масти         | Колонка "действия", 1-я кнопка |
| /horses  | Услуги        | Колонка "действия", 2-я кнопка |
| /prices  | Цены и услуги | Колонка "действия", 3-я кнопка |
