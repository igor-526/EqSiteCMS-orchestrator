После реализации этого плана мы можем указать в CORS только 1 сайт конюшни, а система подразумевает несколько конюшен. Как минимум, dev окружение для сайта
# Что нужно сделать
Я хочу сохранить настройки CORS для тех эндпоинтов, которые требуют авторизации (не по header, а по cookies), чтобы они работали только для CMS фронтенда, а остальные GET запросы для consumer фронтендов работали без CORS. К тому же с внедрением S3 это бесполезно
